This course is for introducing students to basic and advanced concepts of Web technologies like html, css, javascript, reactjs etc. This course is designed for MCA 4th semester students. Every topic is given step-by-step so that you can learn it in a very easy way. Learning HTML to design the structure or layout of your website and style it with CSS, with JavaScript you will be able to create your own interactive and dynamic website.
